Problem Statement:

Write a program in any language to find the maximum and minimum element in an array.

The problem statement is to write a code in any programming langauge to find the maximum and minimum numbers from an array specified numbers are inputted to array. The code is written in C++ langauge. Go through the sample input and output. Variables used in the program are given below. The complexity of the program is mentioned below.

Code:

            #include < bits/stdc++.h >
            int main() 

            {  

                int no;

                cin>> no;

                int arr[no]

                for (i=0;i <no;i++)

                {

                    cin>>arr[i]

                }

                sort(arr,arr+no);

                cout<<"The minimum no. is "<<arr[0]<<"\n";

                cout<<"The maximum no. is "<<arr[no-1]<<"\n";

            }

Variables used in the program:

i, no, arr

Sample Input:

10
56 32 78 54 12 90 34 27 27 43

Sample Output:
The minimum no. is 12
The maximum no. is 90

Complexity:

O(nlogn)
